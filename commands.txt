Layers:
		Session
			Window
				Panes


⌃B ⌃B D 
tmux detach 	-	detach in a nested session 


Creating, detaching, ataching & list:

tmux					-	start a session

tmux new -s <name>		-	start a new session with custom name

tmux a 					-	atach to last used session

tmux a -t <name>		-	atach to a specified session

⌃B D					-	detach from a session

tmux ls 				-	list sessions




Navigation & Killing:

⌃B W 			-	select session, window, and pane

⌃B X 			-	kills the selected/current pane

⌃B &			-	kills the slected/current window

tmux kill-session		-	kill last used session

tmux kill-session -t <name>		-	kill specified session

⌃B [			-	enables scrolling




Windows:

⌃B C 			-	new window

⌃B n 			-	move to next window

⌃B ,			- 	rename current window




Panes:

⌃B %			-	horizonatal split

⌃B "			-	vertical split

⌃B ↑↓→←			-	switch panes with directional arrows

⌃B Q			-	show pane indexes

⌃B Q <index>	-	switch to pane by index
